# Source Registry

本文件记录研究过程中引用的所有资料。每条资料包含完整元信息，便于追溯和验证。

## 格式说明

| 字段 | 说明 |
|------|------|
| ID | 唯一标识符 (S001, S002...) |
| 类型 | paper / documentation / blog / product / patent / report |
| 标题 | 资料原始标题 |
| 作者/机构 | 第一作者或发布机构 |
| 年份 | 发布年份 |
| URL/DOI | 可访问链接 |
| 核心观点 | 一句话概括该资料的核心贡献 |
| 引用阶段 | 在哪些 Phase 文档中被引用 |
| 可信度 | 高/中/低 |
