# 案例研究：AI 眼镜

## 产品定位

视觉感知型个人助手，通过空间理解和主动提醒增强用户的日常认知能力。

## Context 管理链路

```
┌─────────────┐     ┌──────────────┐     ┌─────────────────┐
│  视觉输入    │────→│  场景理解     │────→│  事件/意图判断   │
│  + 音频输入  │     │  (端侧VLM)   │     │                 │
└─────────────┘     └──────────────┘     └────────┬────────┘
                                                   │
                    ┌──────────────┐               │
                    │  空间记忆     │←──────────────┤
                    │  人物记忆     │               │
                    │  习惯记忆     │               ↓
                    └──────┬───────┘     ┌─────────────────┐
                           │             │  主动服务决策    │
                           └────────────→│  是否提醒？      │
                                         │  如何提醒？      │
                                         └────────┬────────┘
                                                   │
                                                   ↓
                                         ┌─────────────────┐
                                         │  输出执行        │
                                         │  语音/AR叠加/震动│
                                         └─────────────────┘
```

## 典型场景

### 场景 A：会议准备提醒

**触发条件**: 用户走向会议室 + 日历显示 10 分钟后开会

**Context Assembly**:
- Current Scene: 走廊，朝会议室方向移动
- Calendar: 10:00 产品评审会
- Episodic Memory: 上次会议用户忘带笔记本
- Procedural: 会议前 5 分钟提醒准备材料

**输出**: "产品评审会 10 分钟后开始，上次你提到要带设计稿。"

### 场景 B：人物识别辅助

**触发条件**: 用户注视一个人超过 2 秒 + 系统识别到已知人物

**Context Assembly**:
- Visual: 人物面部特征匹配
- Semantic Memory: 张工，产品部，上周讨论过 API 对接
- User State: 似乎在犹豫是否打招呼

**输出**: AR 叠加显示 "张工 - 产品部 - 上周聊过 API 对接"

### 场景 C：环境安全提醒

**触发条件**: 检测到用户即将走入车流

**Context Assembly**:
- Visual: 前方有快速移动车辆
- Motion: 用户正在行走，未减速
- Risk Assessment: 高危

**输出**: 震动 + 语音 "注意前方来车"

## 关键设计决策

| 决策点 | 选项 | 推荐 |
|--------|------|------|
| 视觉处理位置 | 全端侧 / 混合 / 全云端 | 混合：端侧快速检测，云端深度理解 |
| 人脸数据存储 | 本地 / 云端 | 纯本地，不上传 |
| 主动提醒频率 | 高频 / 按需 / 极少 | 可学习的自适应频率 |
| AR 显示时长 | 固定 / 自适应 | 根据注视时间自适应 |

## 隐私治理

- 视觉数据默认不持久化，仅保留结构化描述
- 人脸识别仅限用户主动标记的联系人
- 公共场所拍摄需遵守当地法规
- Bystander 检测：有陌生人时降低记录粒度
