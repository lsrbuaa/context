# 阶段 6：研究典型智能硬件应用场景

## 目标

选几个典型硬件形态，拆解完整的 Context 管理链路。

## 场景 1：AI 眼镜

### 典型能力
视觉感知、空间理解、轻量显示、语音交互、主动提醒

### 研究重点
1. 当前看到什么如何进入 working memory？
2. 空间位置和历史兴趣如何触发主动提醒？
3. 眼镜应该什么时候打扰用户？
4. 视觉记忆如何治理隐私？

### Context 链路
```
摄像头/麦克风 → 场景理解 → 事件判断 → 记忆匹配
    → 是否需要提醒？→ 用户可打扰性评估 → 输出决策
```

### 参考研究
- **Memento**: AR 助手把用户查询和时空、活动 context 绑定，并在类似场景下主动召回
- **Alpha-Service**: AI glasses 主动服务系统，拆成 input unit、task scheduling、tool utilization、memory unit 和 output unit

---

## 场景 2：AI 耳机 / 耳戴设备

### 典型能力
实时语音理解、对话辅助、低打扰提示

### 研究重点
1. 如何从背景对话中识别"用户可能需要帮助"？
2. 提示应该多短？
3. 如何避免社交打断？
4. 过去对话记忆如何辅助当前对话？

### Context 链路
```
麦克风 → 语音识别 → 对话理解 → 意图/需求检测
    → 记忆检索(人物/话题) → 是否介入？→ 生成 1-3 词提示
```

### 参考研究
- **LlamaPIE**: 耳戴设备上的 proactive conversation assistant，在不需要用户显式调用的情况下给出 1-3 个词的低打扰提示

---

## 场景 3：AI Pin / 录音挂坠 / 随身记录器

### 典型能力
全天音频记录、会议摘要、待办提取、生活日志

### 研究重点
1. 全天数据如何过滤？
2. 哪些内容应该进入长期记忆？
3. 会议、承诺、关系、地点如何结构化？
4. 如何处理旁人隐私和授权？

### Context 链路
```
全天音频流 → VAD/场景分类 → 事件切分
    → 重要性评估 → 结构化抽取(待办/承诺/人物/地点)
    → 写入 episodic memory → 定期归纳为 semantic memory
```

### 关键设计决策
- 过滤策略：静默/噪声/重复内容丢弃
- 写入阈值：什么事件值得记住
- 隐私边界：旁人语音如何处理
- 存储策略：本地 vs 云端 vs 混合

## 场景 4：桌面机器人 / 家庭机器人 / VLA 机器人

### 典型能力
空间感知、动作执行、物体操作、主动陪伴

### 研究重点
1. LLM context 和 VLA action context 如何分离？
2. 当前环境状态如何进入动作决策？
3. 过去失败动作如何变成程序性记忆？
4. 用户偏好如何影响机器人行动？

### Context 链路
```
视觉+深度+触觉 → 环境状态感知 → 物体/空间理解
    → 任务规划(LLM) → 动作生成(VLA) → 执行 → 反馈
    → 成功/失败记录 → 程序性记忆更新
```

### 双 Context 架构

```
┌────────────────────────────┐
│  High-level Planning       │
│  (LLM Context)             │
│  - 用户意图               │
│  - 任务分解               │
│  - 语义记忆               │
└────────────┬───────────────┘
             ↓
┌────────────────────────────┐
│  Low-level Action          │
│  (VLA Context)             │
│  - 当前视觉观察           │
│  - 关节状态               │
│  - 物体位姿               │
│  - 动作约束               │
└────────────────────────────┘
```

### 参考研究
- **OpenVLA**: 开源 VLA 模型，基于大规模机器人示范数据训练
- **MemoryVLA**: cognition-memory-action 框架，把工作记忆、感知认知记忆库和动作决策结合

---

## 四类场景对比总结

| 维度 | AI 眼镜 | AI 耳机 | AI Pin | 机器人 |
|------|---------|---------|--------|--------|
| 核心 context | 视觉场景 | 对话流 | 音频事件 | 环境+动作 |
| 记忆重点 | 空间记忆 | 对话/人物记忆 | 事件/承诺记忆 | 动作/物体记忆 |
| 主动服务方式 | 视觉叠加提醒 | 低打扰语音 | 事后摘要推送 | 主动执行动作 |
| 隐私核心风险 | 拍摄他人 | 录音他人 | 全天录音 | 家庭场景 |
| 延迟敏感度 | 中 | 极高 | 低 | 极高 |
| token budget | 大 (视觉丰富) | 小 (语音精简) | 中 (摘要型) | 极大 (多模态) |
